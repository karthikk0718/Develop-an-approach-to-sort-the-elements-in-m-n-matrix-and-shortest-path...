🏭 Matrix Sorting & Shortest Path (Mini Project)
📌 Overview

This project implements an approach to:

Sort elements in an m × n matrix
Find the shortest path to a target cell using BFS (Breadth-First Search)

⚙️ Technologies
Python
Tkinter (GUI)
Collections (deque for BFS)

🧠 Working
Matrix elements are flattened, sorted, and reshaped into m × n format
A robot starts from a given cell
BFS algorithm is used to find the shortest path to the target cell
Path is visualized in the grid

🚀 How to Run
Save code as project.py
Run: python project.py
Select start, target, and obstacles
Click Find Path

📊 Features
Matrix sorting
Shortest path calculation
GUI-based visualization
Obstacle handling
👨‍💻 Conclusion

Demonstrates efficient matrix sorting and shortest path finding using BFS in a real-world warehouse scenario.
